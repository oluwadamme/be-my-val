# Be My Valentine? 💌

A romantic interactive website to ask that special someone to be your Valentine.

## Features

- **Envelope Opener** — tap to reveal the love letter
- **"Why I Adore You"** — animated reasons with a bouncing bear
- **Love Calculator** — dramatic meter that always hits 100% (soulmates, obviously)
- **Typewriter Poem** — handwritten-style poem typed out in real time
- **The Big Question** — "Will you be my Valentine?" with Yes/No buttons
- **Runaway "No" Button** — dodges, shrinks, spins, begs, and melts away
- **Grand Celebration** — fireworks, confetti, emoji rain, and a dancing bear
- **Background Music** — romantic music-box melody (Web Audio API, no files needed)
- **Sparkle Cursor Trail** — hearts and stars follow the mouse
- **Valentine's Countdown** — live countdown timer to Feb 14
- **Shooting Stars** — animated streaks across the night sky

## Tech

Single self-contained `index.html` — no dependencies, no build step, no external assets.

- HTML5 / CSS3 (animations, glassmorphism, backdrop-filter)
- Vanilla JavaScript
- Web Audio API (procedural music + sound effects)
- Google Fonts (Dancing Script, Quicksand)

## Deploy

Drop the folder on [Netlify Drop](https://app.netlify.com/drop) for instant free hosting.

## Customize

Edit `index.html` to personalize:

- **Reasons** — update the 6 reasons in the `sec-reasons` section
- **Poem** — change the `poemLines` array in the script
- **Names** — replace "Me" and "You" in the love calculator
- **Celebration message** — edit the text in `sec-yes`
